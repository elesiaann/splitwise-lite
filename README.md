# Splitwise Lite 💸

A simple, lightweight shared expense tracker that works entirely in your browser. No accounts, no servers, no complexity.

## Features

- **Manage Groups:** Create multiple groups for trips, roommates, or events.
- **Track Expenses:** Add expenses with multiple split options (Equal, Exact Amounts, Percentages).
- **Simplify Debts:** Automatically calculates the most efficient way for everyone to settle up.
- **Settle Up:** Record payments between members to keep track of balances.
- **Data Privacy:** All data is stored locally in your browser's `localStorage`. Nothing is ever sent to a server.
- **Export/Import:** Easily backup your data or move it to another device via JSON files.
- **Mobile Friendly:** Responsive design that works great on phones and tablets.

## How to Use

1. **Create a Group:** Give it a name and add at least two members.
2. **Add Expenses:** Enter the description, amount, who paid, and how to split it.
3. **Check Balances:** Go to the "Balances" tab to see who owes what.
4. **Settle Up:** When someone pays another member back, record it using the "Settle Up" button.
5. **Manage Data:** Use the "Export Data" button on the home screen to save a backup of your expenses.

## Tech Stack

- **HTML5 / CSS3** (Custom properties, Grid, Flexbox)
- **Vanilla JavaScript** (ES6+)
- **LocalStorage API** for data persistence
